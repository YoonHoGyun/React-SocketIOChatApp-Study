# React + Vite
⏰ Socket.IO ChatApp<br/>
- 이 프로젝트는 Node.js & Socket.IO를 활용한 간단한 오픈채팅이다.<br/><br/>


🛠 주요 기능
- 닉네임 입력 후 Server접속
- 실시간 채팅
- 사용자가 방에 들어오거나 나갈경우 관리자를 통한 알림
- validation체크

🧪 테스트 방법
- Tool : VSCode
- 서버 구동 : node server.js
- 클라이언트 구동 : npm run dev
- 접속주소 : http://localhost:5173/
